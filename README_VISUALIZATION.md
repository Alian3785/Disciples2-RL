# Pygame-ce визуализация для GridWorld Combat Environment

Этот файл содержит интерактивную визуализацию для среды GridWorldCombatEnv с использованием pygame-ce. Агент перемещается по 10x10 сетке, сражается с врагами по порядку и лечится в специальной клетке.

## Запуск

### Базовый запуск
```bash
python visualize_pygame.py
```

### Запуск с указанием чекпоинта
```bash
python visualize_pygame.py --checkpoint ./checkpoints2/ppo_model_600k_steps.zip
```

### Запуск с настройкой скорости
```bash
python visualize_pygame.py --speed 2.0
```

## Управление

- **SPACE** - Пауза/Продолжить симуляцию
- **R** - Сброс симуляции
- **+** или **=** - Увеличить скорость
- **-** - Уменьшить скорость
- **ESC** - Выход

## Игровая механика

### Цель игры
Агент должен по порядку победить 3 врагов с уровнями 1, 2, 3, которые появляются в случайных позициях на карте. После каждой победы уровень агента повышается, но он становится "раненым" (wounded).

### Действия
- **0-7**: Движение в 8 направлениях
- **8**: Лечение (только в клетке (0,0), снимает wounded статус)

### Правила боя
- **Если wounded**: Немедленное поражение при столкновении с врагом
- **Неправильный порядок**: Поражение при атаке не следующего по порядку врага
- **Уровень >= уровню врага**: Победа, уровень +1, wounded = True
- **Уровень < уровню врага**: Поражение, wounded = True

### Лечение
- Работает только в клетке (0,0) с желтой меткой "H"
- Снимает wounded статус
- Позиция агента не меняется

## Возможности визуализации

### Игровое поле
- **10x10 сетка** с четкой разметкой
- **Клетка лечения** выделена желтым цветом с меткой "H"
- **Координатная сетка** для удобства ориентирования

### Игровые объекты
- **Агент** (зеленый круг с уровнем внутри)
- **Враг 1** (красный круг с уровнем 1)
- **Враг 2** (оранжевый круг с уровнем 2)
- **Враг 3** (фиолетовый круг с уровнем 3)
- **Wounded статус** (красная буква "W" рядом с агентом)

### Информация об игре
- **Уровень агента** и wounded статус
- **Следующая цель** (какого врага нужно победить следующим)
- **Статус успеха** (выполнена ли миссия)
- **Текущий шаг** симуляции
- **Накопленная награда**
- **Текущая скорость** симуляции

### Легенда
- **A - Агент** (зеленый)
- **1 - Враг 1** (красный)
- **2 - Враг 2** (оранжевый)
- **3 - Враг 3** (фиолетовый)
- **H - Лечение** (желтый)

## Особенности среды

### Система спавна врагов
- Враги появляются в **разнообразных зонах** дистанций:
  - Враг 1: ближняя зона (дистанция 1-3 от старта)
  - Враг 2: средняя зона (дистанция 5-9 от старта)
  - Враг 3: дальняя зона (дистанция 12+ от старта)
- Гарантируется **строгое возрастание** дистанций
- **Уникальные позиции** для каждого врага

### Основные исправления:
1. **Правильная загрузка VecNormalize**: Корректное восстановление нормализации наблюдений

- **VecNormalize**: Автоматически загружается из `vecnormalize.pkl` при наличии
- **Monitor**: Обертка для корректного сброса и логирования
- **Синхронизация состояния**: Полное обновление всех параметров агента и врагов после каждого шага

### Награды
- **step_cost**: -0.005 за каждый шаг
- **win_reward**: +0.5 за победу над врагом 1 или 2
- **goal_reward**: +4.0 при победе над 3-м врагом (миссия выполнена)
- **fail_penalty**: -0.35 за поражение
