## Сводка найденного

- №2: В CampaignEnv инвентарь heroitems является неограниченным списком: ёмкость и состояние «полон» не моделируются.
- №4: У общего инвентаря нет адресуемых слотов и действий переноса/обмена; слоты есть только у экипировки.
- №5: У общего инвентаря нет адресуемых слотов и действий переноса/обмена; слоты есть только у экипировки.
- №6: Кампанийная среда хранит один BLUE-отряд с одним походным героем; передачи между героями нет.
- №7: Нет API передачи предмета выбранному юниту; heroitems принадлежат отряду/герою целиком.
- №21: В наборе действий и определений предметов не найден универсальный очищающий предмет.
- №23: Есть автоматическая экипировка/замена артефактов, но отдельного действия снятия артефакта нет.
- №31: Нет отдельной модели владельца предмета и события передачи инвентаря после его смерти.
