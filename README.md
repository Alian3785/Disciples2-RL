So this is the project to train a Superhuman bot for Disciples 2, popular turn-based strategy game. There are 3 stages. First an agent is trained for single player scenarious. Then a more general agent is trained for
maps. Finally an AlphaZero-like agent is trained with self-play to match best Disciples 2 players and beat the champion whoever that might be.

Version if Disciples 2 the agent id trained for is Disciples II: Rise of the Elves (https://store.steampowered.com/app/1630/Disciples_II_Rise_of_the_Elves/)
